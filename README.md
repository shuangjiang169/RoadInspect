# RoadInspect
一个实时道路颠簸程度检测的应用，利用手机加速度计和于第三方的SVC库进行判断处理，svm模型对加速度计三轴采集的数据进行类别判断;

添加了一些辅助功能：定位显示，说明书，注册登录界面;

新手任务,只会拼拼轮子；
